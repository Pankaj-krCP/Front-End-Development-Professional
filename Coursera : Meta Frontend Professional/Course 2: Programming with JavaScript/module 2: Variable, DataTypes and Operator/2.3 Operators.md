- **Logical AND (`&&`)**: Checks if multiple comparisons are true.

  ```javascript
  var currentTime = 10;
  console.log(currentTime > 9 && currentTime < 17); // true
  ```

  - `true && true` â†’ `true`
  - `true && false` â†’ `false`
  - `false && true` â†’ `false`
  - `false && false` â†’ `false`

- **Logical OR (`||`)**: Checks if at least one comparison is true.

  ```javascript
  var currentTime = 7;
  console.log(currentTime < 9 || currentTime > 17); // true
  ```

  - `true || true` â†’ `true`
  - `true || false` â†’ `true`
  - `false || true` â†’ `true`
  - `false || false` â†’ `false`

- **Logical NOT (`!`)**: Flips the boolean value.

  ```javascript
  var petHungry = true;
  console.log(!petHungry); // false
  ```

  - `!true` â†’ `false`
  - `!false` â†’ `true`
